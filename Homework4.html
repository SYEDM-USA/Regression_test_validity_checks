1.2The assumptions for linear regression are the following:

(a)The relationship between response y and the regressors is linear, at least approximately.

(b)The random error term has zero mean and constant variance.

(c)The errors are uncorrelated.

(d)The errors are normally distributed.

1.3The following code constructs scatter plots for response variable vs.the regressors. We note that the plot or y vs.x6 shows a clear linear trend. The y vs.x7 is not as good as the other one, but an approximate linear trend can be observed in this plot as well.

pairs(dat)

1.4The regressor vs.residual plot can help us to check the linearity assumption. The plot is created below. As we scan such plots from left to right by considering thin vertical stripes and notice that in each stripe the vertical average of the residuals is approximately zero, we conclude that the linearity assumption is validated in both plots.

par(mfcol = c(1, 2))
plot(dat$x6, mlr$residuals, 
     main = "Check for linearity \n Residuals vs. x6")
abline(h=0)
plot(dat$x7, mlr$residuals, 
     main = "Check for linearity \n Residuals vs. x7")
abline(h=0)

1.5The residual errors against the fitted values can help us to check the zero mean and constant variance assumption. The plot is created below. As we scan such plots from left to right by considering thin vertical stripes and notice that in each stripe the vertical average of the residuals is approximately zero. We also note that the spread of the data points are approximately the same in all vertical stripes. Hence we conclude that the assumption of zero mean and constant variance is satisfied.

plot(mlr$fitted.values, mlr$residuals, 
     main = "Check for 0 mean and constant var \n  Residual vs. fitted value")
abline(h=0)

1.6The residual errors plotted against the row numbers sorted by regressor variables can help us to check the independence of error assumption. The plot is created below. As we scan such plots from left to right by considering thin vertical stripes we notice that randomly and symmetrically distributed around zero. Hence we conclude that the independence of error assumption is validated by both plots.

par(mfcol = c(1, 2))
row_num <- c(1:nrow(dat))
sort_x1 <- sort(dat$x6, index.return=TRUE)
plot(row_num, mlr$residuals[sort_x1$ix], 
     main = "Check for independence \n Residuals sorted by x6")
abline(h=0)
sort_x2 <- sort(dat$x7, index.return=TRUE)
plot(row_num, mlr$residuals[sort_x2$ix], 
     main = "Check for independence \n Residuals sorted by x7")
abline(h=0)

1.7The normal q-q plot can help us to check the assumption that error terms follow a normal distribution. The plot is created below. The normal q-q plot suggests that the error terms follow a symmetric light tailed distribution. This is not a severe violation from normality.

qqnorm(mlr$residuals)
qqline(mlr$residuals)

1.8The Shapiro-Wilk test can also be performed to check the normality assumption of error terms. The test is performed below. Here the null hypothesis is error terms follow normal distribution. Based on the p-value (greater than \(\alpha = 0.05\)) we conclude that we fail to reject the null hypothesis and hence it is likely that error terms follow normal distribution.

shapiro.test(mlr$residuals)

1.2The assumptions for linear regression are the following:

(a)The relationship between response y and the regressors is linear, at least approximately.

(b)The random error term has zero mean and constant variance.

(c)The errors are uncorrelated.

(d)The errors are normally distributed.

2.3The following code constructs scatter plot for response variable vs.the regressors. We note that the plot or y vs.x shows a curvilinear trend.

pairs(dat)

The regressor vs.residual plot can help us to check the linearity assumption. The plot is created below. As we scan the plot from left to right by considering thin vertical stripes we notice that the vertical average of the residuals is negative at both edges and positive in the middle. Hence we conclude that the linearity assumption is violated.

plot(dat$x, slr$residuals, 
     main = "Check for linearity \n Residuals vs. x")
abline(h=0)

The residual errors against the fitted values can help us to check the zero mean and constant variance assumption. The plot is created below. In this plot as well, as we scan the plot from left to right by considering thin vertical stripes we notice that the vertical average of the residuals is negative at both edges and positive in the middle. Hence we conclude that the assumption of zero mean is not satisfied. The spread of the data points however is approximately the same in all vertical stripes. Thus constant variance is most likely satisfied.

plot(slr$fitted.values, slr$residuals, 
     main = "Check for 0 mean and constant var \n  Residual vs. fitted value")
abline(h=0)

The residual errors plotted against the row numbers sorted by regressor variables can help us to check the independence of error assumption. The plot is created below. As we scan the plot from left to right by considering thin vertical stripes we notice that the vertical average of the residuals is negative at both edges and positive in the middle. Hence we conclude that the independence of error assumption is violated here.

row_num <- c(1:nrow(dat))
sort_x1 <- sort(dat$x, index.return=TRUE)
plot(row_num, slr$residuals[sort_x1$ix], 
     main = "Check for independence \n Residuals sorted by x")
abline(h=0)

The normal q-q plot can help us to check the assumption that error terms follow a normal distribution. The plot is created below. The normal q-q plot suggests that the error terms follow a negatively skewed distribution which is a violation from normality.

qqline(slr$residuals)

2.6Finally for estimating the mean DC output where wind_velocity = 5 mph, we will use the fitted regression line given by \(\hat{ytran} = -1.3585 + 0.7107 x\). Substituting \(x=5\) in this equation we get \(ytran = 2.195\). Now recall, \(ytran = y^2\). Hence \(y = sqrt(2.195) = 1.4815\). Thus the mean DC output where wind_velocity = 5 mph is 1.4815.

Note that, the this problem can also be solved by using the predict function as below. Here we do not need the interval, so we dont worry whether we are using prediction or confidence interval.

Another transformation approach for Question 2

Instead of applying a transformation to the response variable, we can also try to transform x. Here we try an inverse transformation for x by creating a new variable called x_tran. From the pairs() plot we note that the scatter plot of x_tran vs.y is approximately linear. The transformation guide prompted us to try this transformation.

Then we fit a SLR model by regressing y on x_tran. The adjusted \(R^2\) (0.9792) of this model is even better than slr2. We see that there exists multiple good transformations.

Let us know see, how to answer Q2.6 using this model. The fitted regression equation is \(\hat{y} = 2.9789 - 6.9345 xtran\). But recall, \(xtran = 1/x\). Hence we have, \(\hat{y} = 2.9789 - 6.9345 \times (1/x)\). Substituting \(x=5\) in this equation we get \(\hat{y} = 2.9789 - 6.9345 \times (1/5) = 1.592\). Thus the mean DC output where wind_velocity = 5 mph is 1.592 using this model. Note that, this estimate is slightly different from the one obtained from slr2 model.

If we want to use the predict() function to get this estimate we use the following code. Notice that, in this case we created the newDat data frame with x_tran = 1/5.

## Confidence interval for mean y for a given value of x=5 i.e. 1/x = 1/5
newDat <- data.frame(x_tran=1/5)
predict(slr3, newdata = newDat, interval = 'confidence', level = 0.95)
